Unparsable file info

unparsable-file

What it detects

A JavaScript file that failed to parse during analysis. Because its syntax could not be read, the API checks for that file were skipped, leaving a gap in the automated coverage.

It reads the per-file usage records gathered during parsing and reports each file that carries a parse error, passing through the parser's own error text. This is distinct from the api-coverage check, which reports dynamic or aliased access in files that did parse.
