Check the submission for spam error

check-submission-spam

What it detects

This is a human-review item with no automated logic behind it. A reviewer inspects the listing — name, summary, description and screenshots — together with the add-on itself.

The goal is to decide whether the submission looks like spam or carries inappropriate, misleading, or low-effort content. Because this protects the catalogue from abusive listings, a confirmed problem is treated as a hard rejection rather than a soft warning.
