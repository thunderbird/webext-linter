Function constructor usage error

function-constructor

What it detects

Use of the Function constructor — building a function from a string of source code with new Function(...) — in the add-on's own JavaScript. Like eval, it turns a string into running code, which is dynamic code execution and is not allowed.

A single shared scan walks every authored JavaScript source for the dynamic-execution signals and tags each hit by kind; bundled third-party libraries are excluded from that scan. This check reports each hit tagged as a Function-constructor use, one finding per site.
