Trademark violation error

trademark-violation

What it detects

Mozilla's trademark policy forbids certain brand names in an add-on's name. "Firefox", "Mozilla" and "MZLA" are never allowed. "Thunderbird" is only allowed in the trailing form "<name> for Thunderbird".

The check reads the add-on name from the manifest. If the name is a localized __MSG_…__ placeholder, it resolves the real text from every locale file and checks each resolved name. All matching is case-insensitive. The add-on icon is the other trademark vector, but being an image it is left to manual review instead.

When it fires The add-on is told its name uses a registered Mozilla trademark, which is not allowed except in the "<name> for Thunderbird" format.
