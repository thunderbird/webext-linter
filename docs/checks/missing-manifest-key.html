Missing manifest key error

missing-manifest-key

What it detects

Some APIs only work when the manifest declares a particular key — for example, the action API needs an action (or browser_action) key. This check flags calls to such an API when none of the keys it needs are declared.

It shares the same permission analysis as missing-permission. The difference is the remedy: here the fix is to add a manifest key, not a permission. The schema expresses "this API needs a key" as a pseudo-permission of the form manifest:<key>, and the manifest must declare at least one of the named keys.

Decision flow

flowchart TD
