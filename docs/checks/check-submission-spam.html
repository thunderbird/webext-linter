Check the submission for spam manual

check-submission-spam

What it detects

This is a human-review item with no automated logic behind it. A reviewer inspects the listing — name, summary, description and screenshots — together with the add-on itself.

The goal is to decide whether the submission looks like spam or carries inappropriate, misleading, or low-effort content. Like the other manual checks it carries no automated severity — a confirmed problem is raised for the developer from the manual-review section.
