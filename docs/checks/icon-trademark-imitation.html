Check the icon for trademark or logo imitation manual

icon-trademark-imitation

What it detects

This is a human-review item with no automated logic behind it, because it is about an image the automated checks cannot see. A reviewer looks at the add-on's icon.

The question is whether the icon imitates or incorporates the Thunderbird or Mozilla logo. Using those marks on an add-on icon is not allowed under Mozilla's trademark policy.

When it fires The developer is told the icon imitates or incorporates a Mozilla trademark — the Thunderbird or Mozilla logo — which is not allowed, and is asked to use a distinct icon.
